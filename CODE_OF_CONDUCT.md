# Code of Conduct

This project follows the
**[Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**.

Maintainers and contributors are expected to act in accordance with
the Covenant in all project spaces — issues, pull requests,
discussions, chats, and any in-person events related to this
project.

The full text of the Covenant is available at the URL above; please
read it before participating.

## Reporting

Concerns about conduct in any project space can be sent privately
to:

**info@makemytechnology.com**

Reports are handled confidentially. We will acknowledge receipt
within five working days and follow the enforcement guidelines in
the Covenant.

## Scope

This Code of Conduct applies to all project spaces, and also applies
when an individual is representing the project or its community in
public spaces.
