Reset levels (narrowest → widest):
  • Wipe Data — truncate user-data tables (UEs, PDU sessions, IMS regs, …). Schema + bootstrap rows (PLMN, GUAMI) kept. sacore stays up. ≈ 1 s.
  • Remove DB File & Reset to Baseline — delete sacore.db (+ WAL siblings), exit, let docker restart sacore. Cold boot re-runs EnsureSchema + SeedAll from db/seed/baseline.yaml (128 UEs, 3 slices, 5 DNNs, IMS subs). ≈ 5–10 s.
  • Restart sacore — process restart only. DB preserved; recycles every in-memory cache + NF socket. For boot-only config changes. ≈ 5 s.
